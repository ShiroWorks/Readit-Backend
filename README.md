# Readit

Steps to run this project:

1. Run `npm i` command
2. Setup database settings inside `ormconfig.json` file
3. Run `npm start` command

## This app uses the following tech:

- Node/Express
- NextJS
- TypeScript
- TypeORM
- TailwindCSS

## With this app you are able to:

- Register
- Create new community
- Create new post
- Comment
- Arrow voting system
- Upload image
